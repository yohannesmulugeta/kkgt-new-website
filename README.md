# KKGT New Website

A multi-page React/Vite testing website for KKGT Import Export.

## Included

- Cinematic full-screen hero
- Sticky business-area storytelling
- Horizontal coffee-origin scroll on desktop
- Searchable agrochemical catalogue
- Coffee export, commodities, sustainability, quality and contact pages
- Responsive mobile layouts
- Reduced-motion support
- GitHub Pages deployment workflow
- Existing KKGT public image URLs with designed fallbacks

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages

The Vite base path is configured for:

`/kkgt-new-website/`

After the repository is configured to use **GitHub Actions** as its Pages source, every push to `main` deploys automatically.

## Content status

This is a testing website. Company statistics, certificate details, product technical information, staff profiles and contact details must be verified before production launch.
