# KKGT New Website

Professional testing website for KKGT Import Export, published directly through GitHub Pages.

## Live website

https://yohannesmulugeta.github.io/kkgt-new-website/

## Published architecture

The live testing build is a lightweight static single-page website so GitHub Pages can serve it directly from `main / (root)` without a build server.

- `index.html` — published application shell
- `404.html` — internal-route recovery for GitHub Pages
- `standalone/site.css` — responsive design system
- `standalone/site-1.js` to `site-4.js` — pages, routing and interaction logic
- `standalone/image-data-1.js` to `image-data-4.js` — local illustrative test visuals
- `standalone/images/kkgt-logo.svg` — local website logo

## Important content status

This remains a testing website. Contact details, company history, coffee specifications, commodity specifications, agrochemical technical information, certification details, leadership information and all documentary claims must be confirmed by KKGT before an official production launch.

The agricultural illustrations are design assets for testing and are explicitly labelled. They are not presented as photographs of KKGT farms, facilities, employees or completed projects.
